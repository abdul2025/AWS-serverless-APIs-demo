# AWS Serverless APIs Demo

This project demonstrates the usage and consumption of various **AWS Serverless APIs**, built using **AWS CDK (Python)** and deployed via **AWS ECR**.  

---

## Supported API Types

- **REST API** – Standard HTTP-based API endpoints  
- **MQTT** – Lightweight messaging protocol for IoT applications  
- **Webhook** – Event-driven HTTP callbacks  
- **WebSocket** – Full-duplex real-time communication  

---

## Features

- Fully **serverless architecture**  
- Infrastructure defined with **AWS CDK (Python)**  
- Containerized deployment using **AWS ECR**  
- Demonstrates multiple API communication patterns
